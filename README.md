# tcalc
Simple terminal inline calculator

### Install & Usage:
1. Download latest [release](https://github.com/tomaslobato/tcalc/releases/) (.whl file) 
2. In the terminal do `pip install tcalc-X.X.X-py3-none-any.whl --break-system-packages`
3. try it: `calc "2*2+ 3 / 4*5"`
4. keep your calculation inside quotes (""), and don't worry about spaces.
